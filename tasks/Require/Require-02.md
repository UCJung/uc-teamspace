# 요구사항
보고서 취합 기능에 대한 요구사항
## 권한
팀장과 파트장이 보고서 취합을 할 수 있음
## 동작방식
1. 취합할 기간과 대상 인원을 선택 : 전체(모든 인원), 파트선택(선택된 파트의 인원)
    * 취합할 기간 : 2026년 10주차 (3/2 ~ 3/6) (기본값은 이번주)
    * 대상 인원 : 전체 또는 파트를 선택
2. 선택된 인원이 작성한 주간업무 목록 조회
    * 프로젝트(명 + 코드), 팀원(이름 + 파트), 진행업무 (한일), 예정업무 (할일), 비고및 이슈 목록 조회
    * 정렬순서 : 프로젝트 정렬순서 > 팀원정렬 순서
3. 취합 보고서 수정
    * 업무항목 병합 : 같은 프로젝트 내에서는 다른팀원간의 업무를 하나로 병합하는 기능
        * 업무항목 행을 선택하고 병합을 누르면 하나로 병합한다.
        * 진행업무, 예정업무, 비고 및 이슈를 하나의 행으로 표시한다.
        * 팀원에는 합쳐진 원래행의 팀원을 모두 표시한다.
    * 업무수정 : 기존의 진행업무, 예정업무, 비고 및 이슈를 수정할 수 있다.
        * 취합 보고서에서 업무항목별 수정된 내용이 원 작성자의 보고서의 내용을 변경해서는 안된다.
    * 업무삭제 : 유사한 업무 내용의 두개 행이 있을 경우 하나의 행을 삭제할 수 있어야 한다.
    * 업무추가 : 제공하지 않음
4. 취합 보고서 저장
    * 몇주차, 대상 취합보고서로 생성된다. 
        * 전체 선택한 경우 : 2026년 10주차 (3/2 ~ 3/6), 팀명 보고서, 
        * 파트 선택한 경우 : 2026년 10주차 (3/2 ~ 3/6), 팀>파트명 보고서
5. 취합 보고서 제출
    * 모든 내용검토가 끝나면 보고서를 제출한다.
    * 제출된 보고서는 수정할 수 없다.
