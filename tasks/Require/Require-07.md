# 요구사항
월별 근무 시간표 작성의 기능 개선 사항

## 기능에 대한 개요
1차 개발된 월별 근무 시간표의 사용자 검증 피드백 개선사항 및 오류사항

## 기능별 개선 요구사항
### 팀 근무시간 검토
* 파트장 : 파트원의 근무시간을 검토하고 팀장에게 제출, (파트가 없는 사람의 경우 제출 시 팀장에게 제출 됨)
* 팀장 : 전체 제출된 근무신간표를 확인 후 승인 - 일괄승인 기능 제공(목록에서 전체선택 혹은 일부를 선택하여 일괄로 스인)
* 공통기능 : 팀원 제출현황에서 팝업 닫기)
* 필터링 기능 : 컨텐츠 상단 배너영역 : 파트선택(기본값 - 전체) > 연결된 팀원선택,
* 팀원제출현황 : 
  * 팀원목록에 파트 추가 : 성명, (추가)직급, 파트, 직책 (팀원×프로젝트 투입 매트릭스 목록도 동일)
  * 팀원제출현황 타이틀의 우측에 : 총원 Ｎ명, 제출 N명, 미제출 N명 표시 
  * 팀원목록에서 행선택 시 선택된 인원의 근무시간표를 화면전체 사이즈의 레이어팝업으로 표시 (ESC 혹은 우측상단 X 버튼으로 닫기)
  * 액션버튼 행은 우측정렬 고정 너비로 변경 : 행선택 기능(개인근무시간표 조회)과 액션버튼 선택에 간섭하지 않도록 액션버튼행에서는 행선택기능 제외

### 대시보드
주간업무보고 현황과 근무시간표 현황을 모두 볼수 있게 수정
* 상단 배너 영역 : 주간업무보고, 근무시간표, 선택 기능 제공 - 선택에 따라 주차선택(월선택) 하고 아래 컨텐츠도 변경
* 카드영역의 카트를 선택하면 해당 카드 기준으로 목록 필터링
* 주간업무보고 (기본값 전주)
  * 팀원 작성현황 타이틀에 "- 주간업무보고" 추가해서 타이틀 변경
    * 필드 추가 및 순서 : 성명, (추가)직급, 파트, 직책, 이하 기존과 동일
* 근무시간표 (기본값 전월)
  * 팀원 작성현황 집계카드 영역 -  인원 표시 전체, 제출, 미제출, 작성중(임시저장한 수), 미작성
  * 팀원 작성현황 - 근무시간표 로 타이틀 표시
  * 목록부분 - 팀 근무시간표 검토 > 팀원제출현황 표시
