# 요구사항
월별 근무 시간표 작성 기능 개발을 위한 요구사항 1차안

## 기능에 대한 개요
모든 구성원의 월간 일별 프로젝트별 근무시간표를 작성하여 프로젝트별 투입인원의 투입비율에 따른 인건비 원가를 산출하기 위한 기능임

## 주요기능
### 팀원 - 근무시간표 작성
* 월에 한번 모든 구성원 (팀원, 팀장, 파트장 등 전체)은 일단위 근무시간표를 작성함.
* 팀원은 해당원에 휴일근무일, 휴가일 지정할 수 있음
    * 휴일근무일에는 근무시간표를 작성해야함.
    * 휴가일은 작성하지 않음
* 근무시간표 작성 방식
    * 해당일 프로젝트별, 투입시간, 업무방식(외근, 내근, 재택, 출장) : 다수의 프로젝트 입력가능
    * 제약조건 : 1일 근무시간은 8시간, 반차의 경우 4시간

* 제출 제약 조건 : 
  * 해당월에 휴일을 제외한 모든 근무시간을 입력했는지 확인
    * 일별 투입시간 합계가 : 반차나 연차등 휴가가 아님경우 = 8, 반차의경우 = 4 여야함
  * 제출이 완료되면 수정 및 변경을 할 수 없음

### 팀장 - 근무시간표 취합
모든 팀원의 근무시간표 작성을 검토하여 승인하는 기능
월별 팀원별 근무시간표 제출 여부 확인 작성 내용확인 및 승인
* 목록조회 : 팀원의 월간 프로젝트별 투입시간, 투입비율 표시 
  * 열헤더 : 프로젝트명
  * 행헤더 : 팀원
  * CEll : 투입시간, 투입비율
* 좌우 횡스크롤 기능 제공
* 팀전체 통계 조회 : 팀원의 프로젝트별 월간합계 투입비율을 조회함.

### 프로젝트 책임자 
프로젝트에 책임자와 책임부서를 지정할 수 있어야 함.
* 프로젝트 생성요청 - 팀장, 파트장이 요청함.
  * 프로젝트명, 프로젝트명 상세, 책임자, 책임부서를 입력하여 생성함
* 프로젝트 승인 - 관리자는 프로젝트코드를 입력하고 승인함
  * 승인된 프로젝트만 팀에서 프로젝트 관리를 통하여 사용할 수 있음
* 프로젝트 책임자에게는 프로젝트 메뉴 표시
  * 프로젝트 투입현황 (연간) : 담당하는 프로젝트별로 월간 인원 투입현황은 조회함
    * 열헤더 : 해당연도 1월 부터 12월
    * 행 : 프로젝트명 - 투입인원명
    * CELL : 투입시간, 투입비율
  * 프로젝트 투입현황 (월간) : 선택된 월의 프로젝트별 투입현황을 조회함
    * 프로젝트명 , 투입인원명, 투입시간, 투입비율 (프로제트명 rowspan)
    * 월간 투입인원 검토 후 승인 M + 5일이내 승인하지 않을 경우 자동 승인

### 관리자
프로젝트별 투입시간에 대한 모든팀 제출 및 모든프로젝트책임자의 승인 여부를 확인할 수 있어야함.
모든팀과 모든프로젝트에서 승인되면 - 관리자가 최종 승인함

* 현황조회
  * 월단위로 프로젝트별, 인원별, 투입시간 및 투입비율 조회 기능 제공
  * 엑셀 다운로드 기능 제공

# 사용자 데이터 구조 변경
* 모든 사용자는 직위를 가짐 (이사, 부장, 차장, 대리, 사원, 수석, 책임, 선임, 전임 연구원) 
* 특정 사용자는 직책을 가질 수 있음 (팀장, 파트장)
