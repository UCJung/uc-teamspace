# 요구사항
사용자 계정 신청 및 승인 절차를 추가하고 소속된 팀이 없을 경우 팀 생성 요청을 하는 프로세스를 추가하려고 함.

## admin 역할의 추가
시스템적인 기능을 관리하는 역할
* 사용자계정 관리 목록 , 계정생성, 계정신청승인, 상태 수정(신청, 승인, 사용중, 종료)
    * 신청 : 로그인화면에서 계정 등록을 신청한 상태
    * 승인 : 관리자가 계정 등록을 승인한 상태 (승인 시 사용자에게 메일 전송 : 신청이 승인되었습니다. 로그인해서 비밀번호를 번경하세요라는 내용)
    * 사용중 : 사용자가 최초 로그인 시 상태를 변경
    * 사용종료 : 사용자가 퇴사 등으로 사용인 종료된 상태 - 사용자 로그인 금지
* 팀 관리 목록, 팀생성, 팀생성신청 승인, 상태 수정 (신청, 승인, 사용중, 종료)
    * 신청 : 로그인 후 메뉴 최상단에 팀생성 메뉴를 선택하여 팀 생성 신청
    * 승인 : 관리자가 팀 생성을 승인한 상태 - 팀이 생성된고 신청자가 팀장으로 설정
    * 사용중 : 신청자가 로그인하면 표시되는 소소된 팀 목록에서 팀 화면에 최초 접근 사용중으로 표시
    * 사용종료 : 팀화면으로 이동 시 팀설정 메뉴에서 사용종료 선택 시 사용종료

## 로그인 화면에서 계정 신청
* 입력항목 : 성명, 이메일, 비밀번호, 비밀번호 확인, 

## 로그인 후 랜딩화면
* 좌측 메뉴 : 팀
    * 메인 화면 : 
        * 팀검색 기능-팀명으로 검색, 
        * 검색필터 - 전체, 소속팀, 미소속팀, 
        * 팀목록 - 팀명, 팀장, (로그인 사용자가 소속된 팀이 아닌 경우 멤버신청)
        * 팀생성 신청 - 신청하는 레이어 팝업 : 팀명으로 신청
    * 소속팀이 없거나 소속팀이 여러개인 경우 팀 메뉴가 로그인 초기화면
    * 소속팀이 하나인 경우 소속팀 선택 화면이 로그인 초기화면
* 좌측 메뉴 : 로그인 사용자 정보
    * 선택 시 
        * 소속된 팀 목록 표시 - 특정 팀 선택 시 해당 팀 화면으로 이동
        * 비밀번호 변경 메뉴 추가 - 선택 시 비밀번호 변경 레이어 팝업 표시 
    * 비밀번호 변경 레이어 팝업
        * 입력항목 : 현재 비밀번호, 신규 비밀번호, 신규 비밀번호 확인 입력
        * 화면 중안에 표시
* 팀 파트관리 메뉴 (기존)
    * 팀원 목록에 member 신청한 사용자 정보를 표시하고 : 승인 / 거절 기능 추가
    * 승인인 팀구성원으로 추가하고 / 거절 시 목록에 표시되지 않음
    * 한번 거절된 사람이 다시 신청 할 수 있음
* 최초 로그인 시 : 비밀번호 변경 레이어 팝업 오픈 - 비밀번호 변경 후 사용 가능
