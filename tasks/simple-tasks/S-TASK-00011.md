# S-TASK-00011: 근무시간표 월간합계에 프로젝트별 투입비율 표시

> 유형: 기능개선
> 작성일: 2026-03-04

---

## 1. 요청 사항

개인 근무시간표 조회 기능의 하단 합계 부분에 프로젝트별 투입 비율(%)을 같이 표시한다.

---

## 2. 수정 파일

| 파일 | 변경 내용 |
|------|----------|
| `packages/frontend/src/pages/MyTimesheet.tsx` | renderFootRow — 프로젝트별 비율 추가 |
| `packages/frontend/src/pages/TeamTimesheetReview.tsx` | TimesheetPopup tfoot — 프로젝트별 비율 추가 |

---

## 3. 체크리스트

- [ ] MyTimesheet 하단 합계행: 프로젝트별 `Xh (XX%)` 형태로 비율 표시
- [ ] TimesheetPopup 합계행: 동일하게 비율 표시
- [ ] 비율 계산: Math.round(프로젝트시간 / 총시간 × 100)
- [ ] 0h 프로젝트는 비율 미표시
- [ ] 빌드 0 에러
