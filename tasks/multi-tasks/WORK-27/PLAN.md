# WORK-27: 주간뷰 시간 그리드 버그 수정

## 요청사항

사용자 요청 원문:
> - 주간뷰 : 드래그앤 드롭으로 날짜 및 시간을 이동할때 특정일에 드롭하면 그 다음일에 적용됨 (시간은 정상)
> - 주간뷰 : 특정 요일에 있는 카드를 시간을 상하 핸들로 조정이 가능해야 하는데 항하 핸들이 보이지 않아 방법을 모르겠음
> - 주간뷰 : 예정일이 다음주인 경우 다른 주에서로 설정된 작업이 예정업무에 표시됨. 예정업무를 일정미지정작업으로 바꾸고 예정일이 지정되지 않은 업무만 표시

## TASK 구성

| TASK | 제목 | Phase |
|------|------|-------|
| TASK-01 | 주간뷰 DnD 날짜 오프바이원 + 리사이즈 핸들 + 예정업무→일정미지정 수정 | 0 |

## 원인 분석

1. **DnD 날짜 하루 밀림**: CSS grid col = dayIndex + 2 (col 2~8), 하지만 handleDragEnd에서 `dayIndex = col - 1`로 역산 → `col - 2`여야 정확
2. **리사이즈 핸들 안 보임**: `showResizeHandles={span > 1}` 조건 → 1시간 카드에는 핸들 없음. 시간이 설정된 모든 카드에 표시 필요
3. **예정업무 열**: scheduledDate가 이번 주 밖인 작업도 표시 → scheduledDate가 없는 작업만 표시 + 라벨 "일정미지정작업"으로 변경
